# 东大计软智常用网址 - 优化版

> 东南大学计软智学院专用网址导航门户

## 🚀 项目概述

这是一个专门为东南大学计算机科学与工程学院、软件学院、人工智能学院师生设计的常用网址导航网站，提供便捷的校内网站访问入口。

### ✨ 主要功能

- **🔍 智能搜索** - 站内实时搜索，支持关键词高亮
- **📂 分类管理** - 学术、服务、资源、娱乐四大分类
- **⭐ 收藏系统** - 本地收藏管理，持久化存储
- **📊 访问统计** - 实时记录网站访问次数
- **🕒 最近访问** - 智能记录最近访问的网站
- **📱 响应式设计** - 完美适配各种设备
- **⌨️ 快捷键支持** - 丰富的键盘快捷操作


## 🛠️ 技术栈

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript
- **样式库**: Tailwind CSS
- **图标库**: Font Awesome 6.7.2
- **字体**: Google Fonts (Inter)
- **部署**: GitHub Pages
- **存储**: localStorage (前端本地存储)

## 📱 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + K` | 聚焦搜索框 |
| `Ctrl/Cmd + D` | 切换主题 |
| `Ctrl/Cmd + F` | 切换拖拽模式 |
| `Esc` | 关闭模态框 |

## 🔧 本地开发

1. 克隆项目到本地
2. 使用本地服务器打开 `index.html`
3. 推荐使用 Live Server 或 Python 简单服务器

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .
```
